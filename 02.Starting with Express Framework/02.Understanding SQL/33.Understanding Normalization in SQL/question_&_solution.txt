MCQs:


Ques1. What is the main goal of normalization in relational databases?

Ans: To eliminate redundant data and ensure data independencies make sense.


Ques2. Which of the following is true for a table to be in First Normal Form (1NF)?

Ans: It must have only atomic(indivisible) values.


Ques3.  A table is in 2NF if:

Ans: It is in 1NF and has no partial dependencies.


Ques4.  Which of the following is an example of a transitive dependency?

Ans: A -> B, B -> C, therefore A -> C.


Ques5. What does 3NF (Third Normal Form) eliminate?

Ans: Transitive Dependencies.


Ques6. Which of the following is NOT a benefit of normalization?

Ans: Faster query performance in all cases.
